# Snake game auto play using A* Algorithm

just a simple snake game using `pygame` and it is auto played using _Heuristic Searching_ technique named `A* (A-star)`.
it uses cost function f = g + h; where
- g is total cost upto the node n and
- h is the heuristic value from current node to goal node.

Edit the `AI_PLAY` to `False` to play using aswd keys.
